# ULead智链 - 高校科技成果转化金融服务平台

这是一个基于React和Vite构建的现代化网页前端应用程序，专为高校科技成果转化金融服务设计。

## 功能特性

- 用户仪表板
- 项目列表展示
- 投资机会浏览
- 响应式设计

## 技术栈

- React 18
- Vite
- ESLint

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 代码检查

```bash
npm run lint
```
